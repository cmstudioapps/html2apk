Doctor, build e dependencias

doctor

Verifica se Java, Gradle, Cordova e Android SDK estao prontos. Nao gera APK.

build

Cria um projeto Cordova temporario, copia seu app web e gera o arquivo APK em dist.

Dependencias no EXE

O executavel empacota html2apk, a interface e dependencias Node/Cordova. JDK e Android SDK ainda precisam existir na maquina por tamanho e licenca.
